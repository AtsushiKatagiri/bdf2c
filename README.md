# bdf2c

Convert bdf font file to C array sourcecode, convinient for embedded use.

## Usage:
```
bdf2c [-i] [-r 90|180|270] [-d 1] < font.bdf > font.c

options
-i  Invert
-r  Rotate
-d  Array dimension
```

## Exsamples:
